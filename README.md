# 出欠確認システム
- オーケストラ・吹奏楽などの楽団で出欠を簡単に確認することができます.
- 他サービスを使うことなく、Discord内で完結したシステムです。便利!
- 団員は 出席/欠席/早退/遅刻 のリアクションをするだけ。出力も簡単です.
